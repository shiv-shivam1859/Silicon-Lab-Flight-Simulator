# Silicon Lab 🚀
*A web-based 3D aerospace engineering visualizer.*

I am building Silicon Lab as a personal project to bridge the gap between computer science and aerospace engineering. My goal is to eventually create a full flight physics simulator, but right now, it is a lightweight, browser-based 3D parametric visualizer.

## Features
* **Real-time 3D Rendering:** Generates accurate 3D meshes of rocket airframes, nose cones, and fins using Three.js, which updates instantly as parameters change.
* **STL Export:** Ability to export the generated 3D models as `.stl` files for actual 3D printing.

## Tech Stack
* **Frontend:** HTML5, Vanilla JavaScript
* **Styling:** Tailwind CSS
* **3D Graphics:** Three.js
* **Icons:** Lucide
